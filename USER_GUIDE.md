# 客戶關係管理系統 (Customer Manager Rust) - 操作使用手冊

## 1. 系統啟動
*   **執行程式**: 在終端機進入專案目錄，輸入 `cargo run`。
*   **介面說明**: 系統啟動後將開啟一個包含三個區塊的視窗：
    *   **左側面板**: 客戶表單與搜尋功能。
    *   **中央面板**: 客戶資料列表。
    *   **頂部/底部**: 狀態提示與數據統計。

## 2. 客戶管理功能
### 新增客戶 (Create)
1. 在左側面板輸入客戶姓名、電子郵件、聯絡電話及地址。
2. 點擊 **「✨ 點擊新增」** 按鈕。
3. 成功後頂部會顯示綠色提示訊息。

### 查詢客戶 (Retrieve)
1. 在左側下方的 **「🔍 快速搜尋」** 框輸入關鍵字。
2. 系統會根據姓名或電子郵件自動篩選列表內容。
3. 點擊右側的 **「🔄 刷新」** 按鈕可手動更新資料庫狀態。

### 編輯客戶 (Update)
1. 在中央清單中找到目標客戶。
2. 點擊該卡片右側的 **「✏️ 編輯」** 按鈕。
3. 此時左側面板會進入 **「📝 編輯模式」**，並自動填入該客戶資料。
4. 修改內容後點擊 **「💾 儲存修改」**；若不想修改，可點擊 **「🚫 取消」**。

### 刪除客戶 (Delete)
1. 點擊選取清單中的客戶卡片（卡片會變為高亮選取狀態）。
2. 點擊視窗右下角的 **「🗑️ 刪除選取項目」**。
3. 系統會永久刪除該客戶並更新列表。

## 3. 注意事項
*   **資料儲存**: 所有資料均儲存於 `customer.db` 本地資料庫檔案，刪除該檔案將導致所有資料遺失。
*   **必要欄位**: 姓名為必填項目，未填寫將無法完成新增。
*   **字體顯示**: 系統依賴 Windows 的微軟正黑體進行顯示，請確保系統字體庫完整。
